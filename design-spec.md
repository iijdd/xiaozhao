# 小招同学 — 大学生理财陪伴AI搭子 智能体设计文档

> 版本 v1.0 | 2026-05-15 | 招商银行数字金融训练营 AI产品赛道 复赛

---

## 一、产品定位

**一句话**：住在微信/招行App里的大学生AI理财搭子——不教做人，只陪你管好小钱包。

**差异化**：不是理财工具，是"财商室友"。用Z世代网感和行为经济学，把硬核金融概念翻译成校园日常。

**目标用户**：大一大二理财小白，月可支配1000-2000元，只接触过余额宝。

---

## 二、技术方案

基于**Coze（扣子）**搭建纯AI智能体，无外层自定义UI。

| 模块 | Coze能力 | 说明 |
|------|---------|------|
| 对话引擎 | Coze Bot | 大模型驱动对话 |
| 知识检索 | 知识库（A+C混合） | 3篇精写文档 + 5张场景卡片 |
| 结构流程 | 工作流 | 财务体检、赛博把脉周报 |
| 数据持久化 | 数据库（4张表） | 用户档案、消费日志、体检历史、目标进度 |
| 主动触达 | 定时触发器 | 每月1日/每周日20:00/沉默3天/储蓄里程碑 |

## 三、System Prompt

见附属文件 `system-prompt.md`，核心结构：

- **角色定位**：神仙财商室友，金融专业大三，备考CFA一级，有踩坑经历
- **行为准则**：不评判、懂梗、微愿望拦截、降维专业
- **核心技能**：行为经济学降维翻译、情绪价值共情引导、动态风险压力测试
- **服务流程**：情绪与意图识别 → 场景化策略匹配 → 网感化输出（三层结构化：共鸣+分析+行动）
- **硬边界**：不推荐具体产品、不承诺收益、不鼓励借贷投资、拒绝"爹味"说教
- **场景路由**：6类意图自动识别（财务焦虑/投资冲动/消费场景/迷茫求助/炫耀分享/闲聊）

## 四、知识库设计（A+C混合）

### A层：3篇精写安全文档

| 文档 | 字数 |
|------|------|
| 大学生理财入门101（复利原理、心理账户、先储蓄后消费、应急金） | ≤500字 |
| 常见理财产品科普（货币基金、指数基金、定存/国债，校园类比解释） | ≤500字 |
| 投资风险教育（收益风险正相关、分散投资、保本高收益=骗局、跟风陷阱） | ≤500字 |

### C层：5张高频场景卡片

| 场景 | 触发 | 策略 |
|------|------|------|
| 月光急救 | 月底只剩200块 | 共情→拆天预算→建议少点外卖 |
| 第一笔定投 | 有结余想理财 | 风险测评→推荐入门产品→10元体验 |
| 奖学金到手 | 拿到一笔意外收入 | 50%储蓄+30%自我投资+20%庆祝 |
| 跟风冲动 | 同学推荐"稳赚"产品 | FOMO粉碎→极端回撤模拟→观察仓 |
| 消费陷阱 | 满减/限量诱惑 | 沉没成本解释→24h冷静期→反问 |

## 五、工作流设计

### 工作流一：3分钟财务体检

```
触发：用户表达财务焦虑 或 说"体检"
  → 节点1-8：4步对话收集数据（月自由金额/储蓄习惯/理财经验/存钱目标）
  → 节点9：计算健康分(0-100) + 同龄人百分位
  → 节点10：输出结构化体检报告（评分+亮点+黑洞+AI解读+行动建议）
```

### 工作流二：赛博把脉周报

```
触发：每周日20:00定时器 或 用户说"周报"
  → 节点1：查询数据库（expense_log + goal_progress）
  → 节点2：计算恩格尔系数/拦截次数/省钱天数/进度增量
  → 节点3：输出结构化周报（恩格尔系数+拦截统计+目标进度+高光时刻+碎碎念）
  → 兜底：若无数据则输出轻量版+鼓励用户开始记录
```

## 七、特殊场景适配

### 炫耀场景
先共情→计算"炫耀成本"→提出平替讨论

### 迷茫场景
目标拆解法：生活费分3个账户（吃饭60%+学习20%+娱乐20%）

### 沉默唤醒
连续3天未互动 → "宝子你是不是把我忘啦😢 还是最近钱包太鼓不需要我了？"

### 储蓄里程碑
达成50%/100% → 庆祝推送 + 进度分享卡片

## 八、Demo演示路径（Golden Path）

```
用户："我这个月又月光了"
  → 小招共情 + 引导体检
  → 4步对话收集数据
  → 输出体检报告（评分+同龄人排名+AI解读）
  → 发现"奶茶黑洞"：月均300元
  → 建议设"毕业旅行基金"
  → 拿铁因子拦截演示（打车vs单车）
  → 定投模拟：每月300元，毕业变3700元
  → 引导10元体验定投
  → 加入搭子群
```
